# Week 1 - JUnit and Mockito

This folder contains the hands-on exercises for practicing **JUnit 5** and **Mockito** as part of the Cognizant Java Full Stack Engineer training.

## Exercises

### 1. Setting Up JUnit
Create a Maven project, add JUnit 5 dependencies, and write basic calculator tests.

### 2. Mocking and Stubbing
Use Mockito to isolate dependencies and stub method behavior.

### 3. Assertions
Practice core JUnit assertion APIs such as `assertEquals`, `assertTrue`, and `assertThrows`.

### 4. AAA Setup Teardown
Structure tests using the Arrange-Act-Assert pattern and lifecycle methods.

### 5. Verifying Interactions
Verify that collaborators are called the expected number of times with the expected arguments.
